# Search by Department

<small>Returns a list of courses in a department</small>

Searches the CSUF course catalog for courses from a given department.

### Endpoint

```bash
GET /courses/department/{department}
```

### Parameters

| Name         | Description                                                                                           | Example                    |
| ------------ | ----------------------------------------------------------------------------------------------------- | -------------------------- |
| `department` | A department abbreviation **OR** full name. Letters must be correctly capitalized if using full name. | `CPSC`, `Computer-Science` |

### Responses

| Code  | Description                                       |
| ----- | ------------------------------------------------- |
| `200` | A JSON Object containing a list of `CourseObject` |
| `400` | Client Error. Malformed request syntax            |
| `404` | Course not found                                  |
| `405` | The request method was not `GET`                  |
| `429` | Too many requests. Client was rate limited        |
| `500` | An unknown internal server error occurred         |

### Response Example

```json filename="GET /courses/department/CPSC"
{
  "success": true,
  "data": [
    {
        "title": "Object-Oriented Programming",
        "description": "The object-oriented programming paradigm: ...",
        "department": "Computer Science",
        "course_code": "CPSC 121",
        "course_id": 538057
    },
    { ... }
  ]
}
```
